## 🚀 How to Deploy in 3 Steps

### Step 1: Create Project Structure
```
Create folders:
- app/ (put pages here)
- pages/
  - services/ (put service pages here)
  - about.jsx
  - contact.jsx
- components/ (put Layout.jsx here)
- public/
```

### Step 2: Install & Test Locally
```bash
npm install
npm run dev
# Visit http://localhost:3000
```

### Step 3: Deploy to Vercel
```bash
# Push to GitHub first
git add .
git commit -m "Initial"
git push

# Then on Vercel.com
# Click "New Project" → Select repo → Deploy
```

**That's it! Your site is live.**
